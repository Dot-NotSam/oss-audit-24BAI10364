## 📌 Project Overview

This project is a structured audit of **Git**, a distributed version control system.

It explores:

* Open-source philosophy and licensing
* Git’s role in the Linux ecosystem
* Practical Linux usage through shell scripting

## ⚙️ Shell Scripts Overview

| Script | Name                        | Description                                                                                       |
| ------ | --------------------------- | ------------------------------------------------------------------------------------------------- |
| 1      | System Identity Report      | Displays system details such as OS, kernel version, user info, uptime, CPU, RAM, and disk usage   |
| 2      | FOSS Package Inspector      | Checks if Git is installed, shows version, binary path, package manager, and install suggestion   |
| 3      | Disk and Permission Auditor | Analyzes directories for permissions, ownership, disk usage, and checks Git config file           |
| 4      | Log File Analyzer           | Processes log file to count keyword occurrences, calculate percentage, and display recent matches |
| 5      | Manifesto Generator         | Generates a personalized open-source manifesto using user input and saves it to a file            |

<br>

## ▶️ How to Run

### Step 1: Navigate to scripts folder

```bash
cd scripts
```

### Step 2: Give execution permission

```bash
chmod +x *.sh
```

### Step 3: Run scripts

```bash
./1_system_identity.sh
./2_foss_package_inspector.sh
./3_disk_auditor.sh
./4_log_file_analyzer.sh ../logs/sample_log_file.txt
./5_manifesto_generator.sh
```
